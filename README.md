# CNode社区 React版

## 技术栈
```
基于react + react-router-dom + redux + ES6/7 + less 重写react版cnode社区，使用webpack打包
```

### 项目使用

``` bash
# 安装依赖
npm install

# 启动服务
npm start
```

### 功能
- [x] 登录
- [x] 退出
- [x] 消息列表
- [x] 消息提醒
- [x] 个人主页
- [x] 用户信息
- [x] 首页列表，上拉加载
- [x] 页面后退，数据还原
- [x] 主题详情，回复，点赞，收藏
- [x] 页面跳转实现登录拦截
